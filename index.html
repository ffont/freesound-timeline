This is a very simple and stupid hack that gets the most popular sounds in Freesound for a given year and month and plays them all at once (in loop). Set the checkbox on for an alternative "measure" of popularity. This hack was originally implemented in 2011 when I was starting to learn about web technologies. It has been recently updated to use Freesound APIv2 and the Audio object, but besides that the bad code has been left intact. Want to contribute? visit the code repository.
